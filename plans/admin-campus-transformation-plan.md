# 管理后台"校园化"改造计划

## 改造范围

只改造 **管理后台前端**（`hanye-take-out-vue3/`），保留后端 API 和数据表不变。

## 改造前后对比

### 1. 侧边栏菜单（`layout/index.vue:16-50`）

| 位置 | 改造前 | 改造后 |
|------|--------|--------|
| 菜单项 | 菜品管理 | 商品管理 |
| 菜单项 | 套餐管理 | 优惠组合 |
| 菜单项 | 分类管理 | 商品分类 |
| 菜单项 | 员工管理 | 管理员管理 |

### 2. 菜品管理 → 商品管理（`views/dish/`）

| 位置 | 改造前 | 改造后 |
|------|--------|--------|
| 页面标题 | 菜品管理 | 商品管理 |
| 添加页标题 | 添加菜品 | 添加商品 |
| 列表表头 | 菜品名称 | 商品名称 |
| 列表表头 | 菜品分类 | 商品分类 |
| 列表表头 | 菜品价格 | 商品价格 |
| 按钮 | + 添加菜品 | + 添加商品 |
| 表单标签 | 菜品名称 | 商品名称 |
| 表单标签 | 菜品分类 | 商品分类 |
| 表单标签 | 菜品价格 | 商品价格 |
| 表单标签 | 菜品图片 | 商品图片 |
| 表单标签 | 菜品描述 | 商品描述 |
| 弹窗文案 | 确认删除该菜品？ | 确认删除该商品？ |

### 3. 套餐管理 → 优惠组合（`views/setmeal/`）

| 位置 | 改造前 | 改造后 |
|------|--------|--------|
| 页面标题 | 套餐管理 | 优惠组合 |
| 添加页标题 | 添加套餐 | 添加组合 |
| 列表表头 | 套餐名称 | 组合名称 |
| 列表表头 | 套餐分类 | 组合分类 |
| 按钮 | + 添加套餐 | + 添加组合 |
| 表单标签 | 套餐名称 | 组合名称 |
| 弹窗文案 | 确认删除该套餐？ | 确认删除该组合？ |

### 4. 分类管理（`views/category/`）

| 位置 | 改造前 | 改造后 |
|------|--------|--------|
| 页面标题 | 分类管理 | 商品分类 |
| 类型筛选 | 全部/菜品分类/套餐分类 | 全部（去掉类型区分） |
| 添加页 | 类型：菜品分类/套餐分类 | 简化为统一分类 |
| 弹窗文案 | 确认删除该分类？ | 确认删除该分类？ |

### 5. API 引用

只修改前端显示文案，API 路径（`/api/dish`、`/api/setmeal` 等）保持不变。

### 6. Dashboard 统计页面

| 位置 | 改造前 | 改造后 |
|------|--------|--------|
| dishStatistics | 菜品销量统计 | 商品销量统计 |
| setmealStatistics | 套餐销量统计 | 组合销量统计 |

## 不修改的部分

- 后端 Java 代码（实体、Controller、Service、Mapper）
- 数据库表结构
- API 接口路径
- 小程序前端（`my-vue3-project/`）

## 执行步骤

1. **`layout/index.vue`**: 修改侧边栏菜单标题（4处）
2. **`dish/index.vue`**: 页面标题、表格表头、按钮文案、弹窗文案（约10处）
3. **`dish/add.vue`**: 表单标签、页面标题（约8处）
4. **`setmeal/index.vue`**: 页面标题、表格表头、按钮文案、弹窗文案（约10处）
5. **`setmeal/add.vue`**: 表单标签、页面标题（约6处）
6. **`setmeal/components/AddDish.vue`**: 组件内文案（约4处）
7. **`category/index.vue`**: 页面标题、类型筛选项（约4处）
8. **`category/add.vue`**: 表单标签（约2处）
9. **`dashboard/components/dishStatistics.vue`**: 统计标题（约2处）
10. **`dashboard/components/setmealStatistics.vue`**: 统计标题（约2处）
